Class Array

Checks whether the request belongs array and observes the rule.

array('a').ok('a') === false // 'a' is not an array.
array('a').ok(['a']) === true
array('a').ok(['a', 'a']) === true
array('a').ok(['a', 'c']) === false // 'c' do not observe the rule `be('a')`
